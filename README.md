# Netflix-Dashboard

This is an interactive Power BI dashboard analyzing the Netflix dataset.
It provides insights into Netflix movies and TV shows using interactive charts and filters.

##Features

-**Browse by type:**Movie or TV show
**Movie/TV show details:**
 -Release year
 -Rating
 -Short description
 -Genre(s)
 -Director
 -Cast
 -Country of Origin

- **Interactive filters:** Explore specific years, genres, or countries  
- **Visual insights:** See trends in Netflix content over time  

## Tools Used

- Power BI  
- SQL  
- Excel  

## How to View

1. Download the `Netflix_Dashboard.pbix` file.  
2. Open it in Power BI Desktop to explore the dashboard interactively.
